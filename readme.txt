== Fixed and Incremental Table Rates for Every Business Need! ==

Take full control of your WooCommerce shipping with WowShipping's three powerful rate options. From simple fixed rates to advanced hybrid rates, set up shipping that's precise, fair, and fully automated.

= Fixed Rates =

Apply a consistent shipping fee for each weight or quantity range for simplicity and predictability.

**Example:**

- $5 for 1 to 5 KG or Quantity
- $7 for 5.01 to 10 KG or Quantity
- $10 for 10.01 to 20 KG or Quantity

= Incremental Rates =

Charge shipping rates that scale automatically with each additional item or weight.

**Example:**

- $2 for every 1 KG
- $2 for every 3 products

= Fixed + Incremental Rates =

Combine a fixed base fee with incremental charges for larger or heavier orders, enabling you to dynamically offer flexible, fair pricing.

**Example:**

- $5 fixed for up to 10 KG or Quantity, then
- $2 for every 1 KG or Quantity beyond 10

== Combine Multiple Conditions ==

WowShipping takes the concept of table-rate shipping to a whole new level, opening new possibilities. You can add up to 3 condition rows for a single shipping rate and continue it for as many table rates as you need. Additionally, you can add advanced display conditions that apply to the entire table, making it easy to handle any shipping scenario.

**Here is an example:**

If the customer is from the **USA**, orders any product from the Fitness category, the Cart weight is between 0.01 KG to 100 KG, then the shipping cost will be **$12 + $1.5 per KG.**

On the other hand, if the customer is from the **UK** and orders any product from the Fitness category, with a Cart weight between 0.01 KG and 100 KG, the shipping cost will be **$14 + $2 per KG.**

With WowShipping, you can handle this complex scenario effortlessly with a single shipping method.

== Installation ==

= Minimum Requirements =

* WordPress 6.4 or greater
* PHP version 7.4 or greater
* MySQL version 5.6 or greater

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t even need to leave your web browser. 

1. Go to your WordPress Plugin installation menu (Dashboard > Plugins > Add New)
2. In the search field type "WowShipping" and press enter.
3. "Install Now" and then click "Active"

= Manual installation =

For Manual installation, you download our product from WordPress directory uploading it to your web-server via your FTP or CPanel application.

1. Download the plugin and unzip it
2. Using an FTP program or CPanel upload the unzipped plugin folder to your WordPress installation’s wp-content/plugins/ directory.
3. Activate the plugin from the Plugins menu (Dashboard > Plugins > Installed Plugins) within the WordPress admin.

= Upgrading =

Automatic update working great for this plugin. If you customize the plugin make sure to store a backup before update.

Sometimes because of the permalink issue during the plugin update, just go to WordPress > Settings > Permalinks and hit the ‘Save’ button.
